# Home Lab Setup

Documentation of my personal home lab, built for learning cybersecurity concepts hands-on.

## Infrastructure

- **NAS**: TerraMaster (Docker host)
- **Network**: Livebox 5 (Orange)

## Services

- `pihole/` — DNS filtering and ad blocking (Pi-hole v6)

## Goals

- Learn network monitoring and DNS filtering
- Document hands-on cybersecurity practice
- Build toward a SOC analyst skill set
